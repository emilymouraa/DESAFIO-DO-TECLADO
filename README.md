# ⌨️ Desafio do Teclado - Instrutora Emily

Um jogo interativo desenvolvido para auxiliar alunos e entusiastas de informática a dominarem o uso do teclado, desde teclas individuais até atalhos complexos e testes de velocidade.

## 🚀 Funcionalidades

O jogo é dividido em 4 etapas progressivas:
* **Fase 1:** Reconhecimento de teclas especiais (Escape, Tab, Backspace, etc).
* **Fase 2:** Prática de atalhos combinados (Ctrl+C, Ctrl+V, Alt+Tab).
* **Fase 3:** Uso da tecla Shift para símbolos superiores.
* **Fase 4:** Teste de velocidade com frases técnicas sobre hardware e software.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação do conteúdo.
* **CSS3:** Design responsivo, gradientes e animações de interface.
* **JavaScript (Vanilla):** Lógica do jogo, manipulação de eventos de teclado e sistema de pontuação/cronômetro.

## 🎮 Como Jogar

1. Clone este repositório ou baixe o arquivo `.html`.
2. Abra o arquivo em qualquer navegador moderno.
3. Clique em **START** e siga as instruções na tela.
4. Tente completar o desafio com o maior score possível!
